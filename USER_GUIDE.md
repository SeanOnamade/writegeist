### Configure Your OpenAI API Key

Before you can use the AI features, you'll need to set up your OpenAI API key:

1. **Get an API key**:
   - Visit [OpenAI Platform](https://platform.openai.com/api-keys)
   - Sign in to your OpenAI account
   - Create a new API key or copy an existing one

2. **Configure the app**:
   - Open Writegeist Desktop
   - Click on **Settings** in the sidebar
   - Paste your OpenAI API key in the "OpenAI API Key" field
   - Click **Save Settings**
   - The app will automatically restart the backend with your new settings

### Verification

After setting up your API key, verify everything works:
- Go to **Chapter Editor** or **Chapter Ingest**
- Try ingesting a sample chapter
- If successful, you should see extracted characters, locations, and other metadata

3. **Review metadata**: The AI will automatically extract characters, locations, and plot elements
4. **Organize your project**: Use the Project page to keep track of your overall story
5. **View the big picture**: Check the Full Book view to see your complete manuscript

## üîß Troubleshooting

### App Won't Start
- Make sure you have administrative privileges to install software
- Try running the installer as administrator
- Check Windows Defender or antivirus isn't blocking the application

### AI Features Not Working
- Verify your OpenAI API key is correctly entered in Settings
- Check that you have credits available in your OpenAI account
- Make sure you have an internet connection

### Data Not Saving
- The app stores data locally in a SQLite database
- If data isn't persisting, try running the app as administrator
- Contact support if the issue continues

### Performance Issues
- The app includes a local API backend that starts automatically
- If you experience slowdowns, try restarting the application
- Check Task Manager to ensure `writegeist-desktop.exe` is running
